# Simon Game 🌀

A classic Simon memory game built with HTML, CSS, and JavaScript.

## Features
- Random sequence generation
- Player input tracking
- Win/lose logic

## How to Play
1. Open `index.html` in a browser
2. Watch the sequence, repeat it back
3. Progress through levels! 🌀

## Code Structure
- `index.html`: Game UI
- `styles.css`: Styling
- `script.js`: Game logic & sequence handling
